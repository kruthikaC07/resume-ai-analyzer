# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes using ATS-style scoring and provides smart improvement suggestions.

## ✨ Features
- ATS Score Calculation
- AI Resume Suggestions
- Flask Backend API
- Modern Responsive UI
- Resume Keyword Analysis
